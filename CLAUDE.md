# YF-201B Waterflow sensor op XIAO ESP32-C3

## Doel
Pulsen tellen, flow rate berekenen in L/min, totaal volume in Liter.

## Hardware
- **Board:** Seeed XIAO ESP32-C3
- **Sensor:** YF-201B waterflow sensor
- **Signaalpin:** GPIO7 (D5)
